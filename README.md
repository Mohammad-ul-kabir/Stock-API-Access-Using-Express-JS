# Stock API Access Using Express JS

A beginner-friendly backend project built with **Node.js**, **Express.js**, and **MongoDB** to fetch and manage stock market data using external APIs.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Axios

## Features

- REST API for stock data
- External stock API integration
- MongoDB data storage

## Run Project

```bash
npm install
npm run dev
```
